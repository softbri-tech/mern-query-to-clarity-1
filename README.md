# Query to Clarity

Query to Clarity is a full-stack MERN application focused on building a structured knowledge platform with clean backend query processing and a modern frontend architecture.

The project is built step by step as part of a public series, starting with authentication and continuing with article management, filtering, and query-based content workflows.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

## Current Features

- User registration
- User login
- JWT-based authentication
- Protected routes
- Auth context on frontend
- Token persistence
- Backend structured with OOP layers:
  - repository
  - service
  - controller
  - middleware
